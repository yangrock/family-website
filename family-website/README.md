## 网站功能

1. **家谱展示** - 清晰展示家庭三代结构
2. **成员介绍** - 每位家庭成员的详细介绍
3. **家庭相册** - 展示家庭照片（需添加实际照片）
4. **联系信息** - 记录家庭重要日期和联系方式
5. **家庭备忘录** - 可保存重要事项到本地浏览器

## 使用方法

### 在线查看
直接双击 `index.html` 文件在浏览器中打开，或使用以下命令：

```bash
# 在Windows上
start index.html

# 使用Python启动简单HTTP服务器
python -m http.server 8000
# 然后在浏览器中访问 http://localhost:8000
```

### 自定义内容

1. **修改家庭成员信息**:
   - 编辑 `index.html` 文件中的相应部分
   - 更新姓名、角色和描述

2. **添加家庭照片**:
   - 替换 `index.html` 中相册部分的占位符
   - 添加 `<img>` 标签指向您的照片文件

3. **更新样式**:
   - 编辑 `style.css` 文件更改颜色、字体和布局
   - 使用CSS自定义网站外观

4. **扩展功能**:
   - 编辑 `script.js` 文件添加JavaScript交互
   - 可添加更多功能如相册轮播、生日提醒等
